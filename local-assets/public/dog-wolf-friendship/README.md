# Dog And Wolf Friendship

本目录是“每日英语听写”的本地测试样例，按真实本地文件夹方式组织，不作为 TypeScript seed 数据。

## 内容

- `text.txt`：完整英文材料。
- `audio.mp3`：材料音频样例。
- `metadata.json`：样例材料首次初始化分类。

当前材料路径是 `local-assets/public/dog-wolf-friendship/`。分区 `public` 表示公共资源，文件夹名 `dog-wolf-friendship` 用于生成材料 ID 和默认标题。

## 音频对应关系

- `audio.mp3` 对应完整材料。

当前只保留一个音频样例。产品导入时按完整材料导入；有音频就播放，没有音频就作为阅读材料。
