## 🌟 这是干什么的？

每日英语听写是一个为英语学习准备的小程序。

你可以把英文句子、段落或文章导入进去，再配上自己的音频，做成专属听力材料。以后打开就能直接播放、暂停、从头听、看原文、整理分类。

## 🎧 适合谁？

- 想用真实英文材料练听力的人。
- 想把自己收集的英文文本做成听写材料的人。
- 想管理文章、句子和音频，不想散落在各个文件夹里的人。
- 想先本地使用，之后再接云端同步的人。

## 💎 可以做什么？

- 📚 导入英文材料
- 🎵 绑定或替换音频
- ▶️ 播放、暂停、继续、从头开始
- 📝 查看和编辑原文
- 🗂️ 创建分类、移动材料、拖拽排序
- 🧹 删除不需要的材料和空分类
- ☁️ 支持本地使用，也支持云端资料库
