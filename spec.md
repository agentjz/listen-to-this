# 每日英语听写 Spec

`spec.md` 是“每日英语听写”的当前产品事实主干。它只记录已经明确并由当前工程承载的产品定位、业务边界、核心概念、用户路径和验收标准。

## 1. 产品定位

每日英语听写是一个微信小程序 + CloudBase 项目，用于把英文材料和可选音频组织成可听、可读、可新建、可删除、可管理的学习材料。

## 2. 当前主线

- 用户可以使用本地接口，不依赖 CloudBase。
- 用户可以使用云端接口，数据进入 CloudBase。
- 本地接口和云端接口分离，不自动同步。
- 用户可以新增、重命名和删除普通分类，并把未归类材料中选中的一个或多个材料导入指定分类。
- 公共资源是只读分类，普通用户不能复制、编辑、移动、删除、排序其中的材料。
- 资源页提供全部资源虚拟入口，聚合展示当前可见材料，但不写入本地缓存或云端数据库。
- 资源页提供搜索框和搜索按钮；输入时显示命中列表，点击结果或搜索按钮会进入材料所在分类并高亮材料。
- 未归类材料一直显示在分类列表底部，不能删除，不能重命名。
- 删除普通分类时，该分类内材料自动迁移到未归类材料。
- 新建内容按完整材料保存。
- 新建材料时可以只保存文本，也可以同时从微信会话文件选择音频。
- 导入和编辑采用手动保存；输入时只更新页面临时状态，点击保存才写入数据。
- 材料有音频就播放。
- 材料没有音频就作为阅读材料。
- 材料页是主要操作界面，支持直接播放、暂停、继续、从头播放、进度展示、拖动跳转、查看原文和管理删除。
- 普通模式点击材料进入原文查看；管理模式通过编辑入口修改标题、正文和音频。

## 3. 核心概念

### 3.1 资料库

资料库是材料来源的最高层组织。

当前资料库类型：

- 本地资料库：保存在当前设备的小程序缓存中。
- 云端资料库：保存在 CloudBase 中。
- 系统库/通用听力：后续由管理员维护的云端资料来源。

当前界面中，资料库也承担用户自定义分类职责。用户可以新增和重命名普通分类，把材料移动到其他分类，并在分类内长按拖拽调整材料顺序。

用户可以删除普通自定义分类。删除普通分类时，该分类内材料自动迁移到未归类材料。未归类材料一直显示在分类列表底部，不能删除，不能重命名。

公共资源是只读分类。普通用户只能查看和播放公共资源里的材料，不能复制、编辑、移动、删除或排序。

全部资源是虚拟入口，用于聚合展示当前可见材料。它不属于 `SourceLibrary`，不进入本地缓存、生成文件或云端数据库。

### 3.2 材料

材料是产品的基本学习单位，可以是一句话、一段话或一篇文章。

材料包含：

- 名称：列表和听力入口展示。
- 英文内容：用于原文展示和阅读。
- 音频数量：用于判断是否可以直接播放。
- 排序值：用于分类内长按拖拽排序和稳定展示。

### 3.3 听力音频

听力音频是材料可播放的音频文件。音频来源包括：

- 用户上传音频。
- 云端生成音频。

用户可以在导入材料时从微信会话文件选择音频，也可以在原文详情页替换材料音频。替换音频后，材料使用新的当前音频记录。

上传音频支持：

- `mp3`
- `m4a`
- `wav`

当前不做整篇音频自动对齐，不做音频自动切句。

## 4. 用户路径

### 4.1 本地使用

1. 用户打开小程序。
2. 用户进入本地资源。
3. 用户看到本地资料库和材料。
4. 用户在材料页直接播放有音频的材料，或点材料标题查看原文。
5. 用户进入管理模式，可以新建材料、从未归类材料中选择一个或多个材料导入到普通分类、移动或删除自己的材料。
6. 用户可以编辑自己在本地创建的材料，并替换材料音频。

### 4.2 云端使用

1. 用户打开小程序。
2. 用户进入云端资源。
3. 小程序通过 CloudBase 读取云端资料库和材料。
4. 用户在材料页直接播放有音频的材料，或点材料标题查看原文。
5. 用户进入管理模式，可以新建材料、从未归类材料中选择一个或多个材料导入到普通分类、移动或删除自己的云端材料。
6. 用户可以编辑自己的云端材料、上传替换音频，或通过已配置的 TTS 生成音频。

当前工程包含：

- 微信小程序 TypeScript 工程。
- Vant Weapp npm 依赖。
- 本地资源和本地新建。
- 云端资源和云端新建。
- 材料保存、读取和删除。
- 材料编辑和音频替换。
- 分类新增、重命名、删除并迁移材料、材料移动分类和分类内长按拖拽排序。
- 全部资源虚拟入口和资源页材料搜索。
- 首页、资源、材料、原文详情、新建材料、导入已有材料页面。
- 本地测试样例 `local-assets/dog-wolf-friendship/`。
- 自动测试、类型检查、乱码检查、坏命名检查和页面入口检查。

当前工程不包含：

- 已部署云环境。
- 已创建真实数据库集合。
- 已配置真实腾讯云 TTS 密钥。
- 已内置大批系统材料。
- 微信开发者工具中的 `miniprogram_npm/` 构建产物。

当前产品不做：

- 自动分句。
- 逐句播放。
- 听写输入和自动评分。
- 音频导出。
- 语法教学。
- 语法分析。
- 口语评分。
- 发音纠错。
- 作文批改。
- AI 老师聊天。
- 社交社区。
- 课程商城。
- 复杂考试题库。
- 多级分类体系。
- 材料市场。
- 自动联网查词。
- 整篇音频自动对齐。
- 音频自动切句。

## 7. 数据边界

当前已确认的数据对象：

- `SourceLibrary`：资料库。
- `Material`：材料。
- `ListeningAudio`：听力音频。
- `DictionaryEntry`：词义记录。

本地数据保存在当前设备或微信开发者工具的小程序缓存中，当前缓存键为 `listen.localData`。云端数据保存在 CloudBase 中。本地和云端不自动同步。

`local-assets/` 是仓库级测试样例目录，不是小程序运行时写入目录。用户在小程序里新建的材料不会自动生成或修改 `local-assets/` 文件。

`local-assets/<folder-id>/` 中的 `<folder-id>` 是样例材料唯一标识，并用于派生默认标题。文件夹名使用小写字母、数字和连字符。样例材料文件使用固定通用命名：`text.txt`、`audio.mp3|m4a|wav`。`metadata.json` 只用于样例材料首次初始化分类，不记录用户运行时移动分类结果。

## 8. 验收标准

- 首页能进入本地资源、本地新建、云端资源、云端新建。
- 本地资源无需云环境即可显示样例材料。
- 材料页支持查看原文、直接播放、管理导入、编辑和删除。
- 导入已有材料页支持从未归类材料选择一个或多个材料、全选、取消选择和导入已选。
- 材料页支持播放、暂停、继续、从头开始、进度展示和拖动跳转。
- 资源页支持新增分类、重命名普通分类、删除普通分类并迁移材料；未归类材料一直置底展示且不可删除、不可重命名。
- 资源页支持搜索当前可见材料；输入时显示结果列表，点击结果或搜索按钮进入材料所在分类并高亮材料。
- 全部资源入口聚合展示当前可见材料，不作为真实分类保存。
- 公共资源只读，普通用户不能编辑、移动、删除、排序或复制公共材料。
- 材料管理模式支持移动分类和长按拖拽排序。
- 详情页支持查看模式和编辑模式；查看模式只显示原文，编辑模式支持修改标题、正文和替换音频。
- 新建材料页支持从微信会话文件选择可选音频，并在保存材料后绑定到该材料。
- 新建材料页和详情编辑页只在点击保存后写入标题和正文。
- 没有音频的材料作为阅读材料存在，不阻塞导入和查看。
- `npm.cmd run verify` 通过。
- `npm.cmd run build` 能生成微信开发者工具运行所需 JS。
